# Reflex Test

反射神経を測定できる Web アプリです。<br>
ゲームの前にやるとウォーミングアップになっていいと思います。

画面や音の変化に対してどれだけ素早く反応できるかを、シンプルな操作で測定できます。<br>
自分自身の反応速度を気軽に計測・記録できるアプリとして開発しました。

リンク:
[反射神経テスト](https://reflex.jinsei-makino.com/)

## Features

* 視覚反応テスト
* 聴覚反応テスト
* 反応時間の計測
* ターゲット円の色変更
* 背景色の変更
* ターゲット円サイズの変更
* PC / スマートフォン対応

※ ターゲット円が表示された瞬間にクリックすることで、反応時間を測定します。

## Tech Stack

* HTML
* CSS
* JavaScript
